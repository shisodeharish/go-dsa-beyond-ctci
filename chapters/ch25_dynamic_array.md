# Chapter 25 – Dynamic Arrays

## Problem 25.1 – Core Dynamic Array
Implements:
- Append
- Get / Set
- PopBack
- Size

## Problem 25.2 – Extras
- Insert(i, x)
- Remove(x)
- Pop(i)
- Contains(x)

## Complexity Table

| Operation    | Time        | Space |
|--------------|-------------|-------|
| Append       | Amortized O(1) | O(n) |
| Get/Set      | O(1)        | O(1) |
| PopBack      | O(1)        | O(1) |
| Insert       | O(n)        | O(n) |
| Pop(i)       | O(n)        | O(n) |
| Remove(x)    | O(n)        | O(n) |
| Contains(x)  | O(n)        | O(1) |
